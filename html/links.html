Links

FLUIDCAL

http://www.ruhr-uni-bochum.de/thermo/Software/Seiten/Fluidcal-eng.htm

Commercial program enables the user to calculate more than 25 different thermodynamic properties of more than 70 substances.

Good list of references to papers with EoS of particular simple fluid is attached to the description.

NIST Chemistry WebBook

http://webbook.nist.gov/chemistry/

Huge database of pure components properties.

FPROPS

http://ascend4.org/FPROPS

FPROPS is a free open-source C-based library for high-accuracy evaluation of thermodynamic properties for a number of pure substances. It makes use of published data for the Helmholtz fundamental equation for those substances.

CoolProp

http://coolprop.sourceforge.net/

CoolProp is an open-source, free property database that includes pure fluids, pseudo-pure fluids, and humid air properties. It is designed to be trivially simple to use from the Python programming language, and possible to use from other languages as well.

VLE using Java

http://vle-java.sourceforge.net/

VLE is a program written in java to simulate the thermodynamic systems containing liquid and vapour. It can determine various system properties like bubble point, dew point, etc. It is designed to be cross-platform compatible and extendable.

List of chemical process simulators

http://en.wikipedia.org/wiki/List_of_chemical_process_simulators

List of chemical process simulators at wikipedia. Could be a good strating point for the consequent research.
