Volume of Compressible Liquid

General Information

According to Dymond and Malhotra (1988), in 1888 Tait established the following equation involving volume of compressible liquids: \[ \frac{V_l - V}{V_l} = C \log\left[(B + P)/(B + P_l)\right] \] where:

  • $V$ — volume
  • $P$ — pressure
  • subscript $_l$ refers to the low-pressure
  • $B$ and $C$ are constants to be defined

Integrated version of this equation could be written as: \[ V = V_l \left(1 - C \ln \frac{B + P}{B + P_l}\right) \] where $V_l$ and $P_l$ now stand for reference volume and pressure.

Pure Liquids

Constants

In case of pure liquids, following relations could be applied: \[ B = P_c \left[ -1 + a (1 - T_r)^{1/3} + b (1 - T_r)^{2/3} + d (1 - T_r) + e (1 - T_r)^{4/3}\right] \\ e = \exp(f + g \omega_0 + h \omega_0^2) \\ C = j + k \omega_0 \] where:

  • $P_c, [Pa]$ — critical pressure
  • $T_r = \frac{T}{T_c}, [0]$ — reduced temperature
  • unknown coefficients $a, \dots, k$ could be found in Thomson, Brobst and Hankinson (1982)
  • $\omega_0, [0]$ — optimized value of the acentric factor based on the SRK equation

Volume

We pick $V_l$ to be saturated volume $V_s$. Following estimation procedure could be applied: \[ V_s = V^{\circ} V_R^{(0)} \left[1 - \omega_0 V_R^{(\delta)}\right] \\ V_R^{(0)} = 1 + a (1 - T_r)^{1/3} + b (1 - T_r)^{2/3} + c (1 - T_r) + d (1 - T_r)^{4/3} \\ V_R^{(\delta)} = \left[e + f T_r + g T_r^2 + h T_r^3 \right] / (T_r - 1.0001) \] where:

  • $V_R^{(0)}, [0]$ — spherical molecule function, relation holds for $0.25 < T_r < 0.95$
  • $V_R^{(\delta)}, [0]$ — needed for deviation function factor, relation holds for $0.25 < T_r < 1.0$
  • $T_r = \frac{T}{T_c}, [0]$ — reduced temperature
  • unknown coefficients $a, \dots, h$ could be found in Hankinson and Thomson (1979)
  • $V^{\circ}, \left[\frac{\text{m}^3}{\text{mol}}\right]$ — characteristic volume, required for each pure compound
  • $\omega_0, [0]$ — optimized value of the acentric factor based on the SRK equation

Characteristic volume $V^{\circ}$ could be either computed with correlation formula presented in Hankinson and Thomson (1979) or looked up from the table ibid. Acentric factor is supposed to coincide with one in Soave-Redlich-Kwong equation.

Pressure

We pick $P_l$ to be saturated pressure $P_s$, which we could compute via Antoine-type equation.

The following schema are due to Poling et al. (2001), description of Riedel equation. \[ \ln P_s = A - \frac{B}{T_r} + C \ln T_r + D T^6_r \] where:

  • $A = -35Q$
  • $B = -36Q$
  • $C = 42Q + \alpha_c$
  • $D = -Q$
  • $Q = K(3.758 - \alpha_c)$
  • $\alpha_c = \frac{3.758 K \psi_b + \ln \left(P_c/1.01325\right)}{K \psi_b - \ln T_{br}}$
  • $\psi_b = -35 + \frac{36}{T_{br}} + 42 \ln T_{br} - T_{br}^6$
  • $T_{br}, [0]$ — reduced boiling temperature
