Fluid PVT

About

This resource is intended to provide necessary information for performing PVT simulation and prediction of some physical and thermodynamical properties of hydrocarbon mixtures. For convenience of the user links to the original papers and sources are provided. All found resources are attached as well.

Simultaneously it serves as comprehensive and interactive documentation for the internal model. Every module is provided with the link to the corresponding description page. The aim of such a documentation are both the simplification of maintaining of the model structure and facilitation of the process of involvement of new person into the subject.

Disclaimer

This information provided "as is". This compilation was done for the simulation needs. It wasn't tested by implementation of some real world engineering application constructed based on data, computed by observed algorithms. If you are interested in using it for a real-world engineering problems, please, do a double check with original sources.

List of topics

  • Characterization of the vapor-liquid equilibrium, description of the model and computational approaches.

  • Description of the Peng-Robinson equation and formulas for different properties, derived from it.

  • On computation of the sonic speed in two phase mixtures.

  • On computation of the heat capacities.

  • On computation of the volume of compressible liquid.

  • Selection of experimental data sources with short description.

  • All notations and references used and cited over the description. Majority of references is in English, only few is in Russian.

  • Links to the useful resources.
